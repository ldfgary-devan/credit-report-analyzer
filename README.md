# credit-report-analyzer

基于 Dify Chatflow + 通义千问 API 构建的征信报告结构化分析工具。
专为银行客户经理设计，用于快速提取、分析、风险预警征信报告内容。

## 两个工具说明

### 工具 A — 征信分析（生产版）
- **用途**：用户上传征信报告 → 输出结构化分析报表
- **当前版本**：v1.1
- **核心能力**：页码驱动数据对齐、字段隔离提取、风险预警、数据质量自评
- **部署平台**：Dify Chatflow

### 工具 B — 规则性评判（调试版）
- **用途**：对照工具A的输出结果 + 原始征信报告，逐条验证是否符合预设规则
- **当前版本**：evaluator_v1.0
- **使用场景**：Prompt 迭代调试时启用，生产环境断开连接
- **连线方式**：原始征信报告 + 工具A输出结果 → 同时接入工具B
